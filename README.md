Ce projet est réalisé en Javascript et un peu HTML dans l’environnement de développement intégré Visual Studio Code dans windows.

Le développeur utilise node.js, la version est 18.18.2.

La libraire utilisée est Vega-Lite. Il a été introduit dans le projet et ne nécessite ni téléchargement ni installation.

Veuillez utiliser un ordinateur connecté à Internet et utiliser Edge ou Google Chrome pour ouvrir la page Web.

projet: Application web de réorganisation des visualisations dans un tableau de bord

--------------------------------------------------------------------------------------------
Démarrez d’abord le fichier index.html.

Entrez le nombre de graphiques et d’itérations dans la page Web ouverte, puis cliquez sur le bouton Envoyer et attendez que les résultats apparaissent.

Utilisez le bouton Actualiser pour actualiser la page. Lorsque vous souhaitez le faire plusieurs fois, veuillez d’abord actualiser la page, puis saisir de nouveaux numéros.
